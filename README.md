# Card Preview Application

## Overview
The Card Preview Application is a simple front-end project that demonstrates how to dynamically display content using reusable card components. The application allows users to switch between different layout views and interact with cards to view additional details in a popup modal.

This project focuses on practicing modern HTML structure, CSS styling, and JavaScript DOM manipulation.

---

## Features
- Toggle between **grid view** and **column view**
- Dynamic card rendering using HTML `<template>` elements
- Popup modal for displaying detailed information
- Responsive layout for different screen sizes
- Clean separation of HTML, CSS, and JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- HTML Templates
